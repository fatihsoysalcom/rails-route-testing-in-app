# Rails Route Testing In App

This example simulates Rails' routing mechanism in a single Ruby file. It demonstrates how to define routes and then directly test them within the application code without needing a full Rails server or separate testing framework.

## Language

`ruby`

## How to Run

Save the code as `rails_route_lab.rb` and run it using `ruby rails_route_lab.rb`.
